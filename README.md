# Amazon Best Selling Books Analysis

## Project Overview

This project analyzes the Amazon Best Selling Books dataset using Python and various data analysis libraries. The goal is to explore book ratings, prices, reviews, and category trends through data visualization and exploratory data analysis (EDA).

## Tools and Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Analysis Performed

* Data Cleaning and Preprocessing
* Fiction vs Non-Fiction Analysis
* Book Rating Analysis
* Price Distribution Analysis
* Category-wise Comparisons
* Data Visualization using Charts

## Key Insights

* Compared trends between Fiction and Non-Fiction books.
* Identified patterns in book ratings and reviews.
* Analyzed pricing distribution of books.
* Visualized category trends using charts and graphs.
